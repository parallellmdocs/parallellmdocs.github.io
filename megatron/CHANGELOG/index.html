    NVIDIA Megatron Core 0.13.0

    • 支持在优化器状态中使用 bf16 dtype，以配合 TransformerEngine 的精度感知优化器
    • MoE
    • 功能：
      • 通过自定义流水线布局（--pipeline-model-parallel-layout）支持灵活的非对称虚拟流水线并行
      • 支持向 MoE 模块传入自定义并行分组
      • 为 MoE 模型提供 Hybrid Shard Data-Parallel 支持（--num-distributed-optimizer-instances）
      • 支持 DeepSeek-V3 结合 Expert Parallel 与自定义 FSDP 的训练
      • 为 Multi-Token-Prediction 引入 FP8 支持
    • 内存优化
      • 细粒度重计算以降低激活显存开销（--recompute-modules 搭配 --recompute-granularity selective）
      • 通过将概率乘法从反向重排列移至 GroupedMLP 的激活函数，提高 token permutation 的内存效率
    • 性能优化
      • MLA RoPE 融合算子与 YARN 嵌入缓存
      • 通过填充路由映射优化 MoE 模型的 FP8 padding
    • 缺陷修复：
      • 修复在启用 expert_bias 或 group limited routing 时的辅助损失计算（load_balancing_loss 数值相较旧版本会发生变化）
      • 修复 MLA 的打包序列支持
    • 已知问题：
      • MTP 暂不兼容灵活流水线布局，预计在 MR !3594 修复
      • TP2 下 MTP 的收敛问题，预计在 MR !3594 修复
