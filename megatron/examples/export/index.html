    Megatron Core 导出

    该模块用于将 Megatron Core 模型导出到不同的推理框架。目前支持 TRT-LLM，未来将陆续支持 VLLM 等。

    量化与导出

    可参考 TensorRT Model Optimizer 的示例，先执行训练后量化，再导出为类似 HF 的检查点，以便部署到 TensorRT-LLM、vLLM、SGLang 等框架。

    TRTLLM 导出

    如只需导出为 TRT-LLM 检查点，请参考 trtllm_export 中的说明。
