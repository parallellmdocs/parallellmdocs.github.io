    # TensorRT Model Optimizer 集成示例 [TensorRT Model Optimizer](https://github.com/NVIDIA/TensorRT-Model-Optimizer) | [本地环境示例](#本地环境快速入门) | [配置说明](ADVANCED.md#learn-more-about-configuration) | [Slurm 示例](ADVANCED.md#slurm-examples) | [推测式解码](speculative.md) | [高级主题](ADVANCED.md)

    TensorRT Model Optimizer（ModelOpt, 包名 nvidia-modelopt）为 NVIDIA 硬件提供端到端模型优化，包括量化（真实或模拟）、稀疏化、蒸馏、剪枝、神经架构搜索及推测式解码。

    主要特性

    • 支持从 Hugging Face 预训练检查点一键转换。
    • 兼容各种模型并行方式（TP、EP、ETP、PP）。
    • 导出统一检查点，可直接用于 TensorRT-LLM、vLLM、SGLang。

    支持矩阵 {模型}×{特性}

    模型 (conf/) 量化 EAGLE3 剪枝（仅 PP） 蒸馏
    moonshotai/Kimi-K2-Instruct ✅ ✅ - -
    Qwen/Qwen3-{30B-A3B, 235B-A22B} WAR ✅ - -
    Qwen/Qwen3-{0.6B, 8B} ✅ ✅ ✅ ✅
    deepseek-ai/DeepSeek-R1 ✅ ✅ - -
    meta-llama/Llama-{3.1-8B, 3.1-405B, 3.2-1B}-Instruct ✅ ✅ ✅ ✅

    本地环境快速入门

    通过 PyPI 安装 nvidia-modelopt： 

    pip install -U nvidia-modelopt
    也可从 源码 安装体验最新功能。

    ⭐ NVFP4 量化、量化感知训练与模型导出

    设置预训练检查点路径 ${HF_MODEL_CKPT}，以及用于保存 Megatron 分布式检查点的 ${MLM_MODEL_SAVE}。若需导出供 TensorRT-LLM、vLLM、SGLang 使用的 Hugging Face 样式量化检查点，请将 ${EXPORT_DIR} 传递给 export.sh。

    📙 注意： ModelOpt 支持多种量化格式。默认采用模拟低精度（fake-quant），适用于算力大于 80 的 GPU。真实低精度权重（如 E4M3、E2M1）与低精度计算（如 FP8Linear）也受支持，具体取决于 GPU 能力。详见 高级主题。

    \
    TP=1 \
    HF_MODEL_CKPT=<pretrained_model_name_or_path> \
    MLM_MODEL_SAVE=/tmp/Llama-3.2-1B-Instruct_quant \
    ./quantize.sh meta-llama/Llama-3.2-1B-Instruct nvfp4

\
    PP=1 \
    HF_MODEL_CKPT=<pretrained_model_name_or_path> \
    MLM_MODEL_CKPT=/tmp/Llama-3.2-1B-Instruct_quant \
    EXPORT_DIR=/tmp/Llama-3.2-1B-Instruct_export \
    ./export.sh meta-llama/Llama-3.2-1B-Instruct

    ❗ 重要： 脚本的第一个位置参数（如 meta-llama/Llama-3.2-1B-Instruct）用于匹配 conf/ 内的配置。预训练检查点需提前下载并通过 ${HF_MODEL_CKPT} 指定。

    加载保存的分布式检查点后，可继续执行推理（生成或评估）或训练（SFT/PEFT）。更多细节参见 高级主题。设计说明正在撰写中。

    \
    TP=1 \
    MLM_MODEL_CKPT=/tmp/Llama-3.2-1B-Instruct_quant \
    ./generate.sh meta-llama/Llama-3.2-1B-Instruct

\
    TP=1 \
    MLM_MODEL_CKPT=/tmp/Llama-3.2-1B-Instruct_quant \
    ./mmlu.sh meta-llama/Llama-3.2-1B-Instruct

\
    TP=1 \
    MLM_MODEL_CKPT=/tmp/Llama-3.2-1B-Instruct_quant \
    ./finetune.sh meta-llama/Llama-3.2-1B-Instruct

    ⭐ 在线 BF16 EAGLE3 训练

    在线 EAGLE3 训练会同时加载目标模型（冻结）与草稿模型，在训练过程中动态生成所需 hidden_states。系统会定期在 MT-Bench 提示上评估接受长度（AL，越高越好）。使用同一个 export.sh 即可导出供部署使用。

    \
    TP=1 \
    HF_MODEL_CKPT=<pretrained_model_name_or_path> \
    MLM_MODEL_SAVE=/tmp/Llama-3.2-1B-Eagle3 \
    ./eagle3.sh meta-llama/Llama-3.2-1B-Instruct

\
    PP=1 \
    HF_MODEL_CKPT=<pretrained_model_name_or_path> \
    MLM_MODEL_CKPT=/tmp/Llama-3.2-1B-Eagle3 \
    EXPORT_DIR=/tmp/Llama-3.2-1B-Eagle3-Export \
    ./export.sh meta-llama/Llama-3.2-1B-Instruct

    更多示例（如 moonshotai/Kimi-K2-Instruct 的 Slurm 作业）可参考 高级主题。

    ⭐ 剪枝

    剪枝入门与参数配置详见 ModelOpt 剪枝文档。

    目前支持 GPT 与 Mamba 模型，参数包括： - TARGET_FFN_HIDDEN_SIZE - TARGET_HIDDEN_SIZE - TARGET_NUM_ATTENTION_HEADS - TARGET_NUM_QUERY_GROUPS - TARGET_MAMBA_NUM_HEADS - TARGET_MAMBA_HEAD_DIM - TARGET_NUM_LAYERS - LAYERS_TO_DROP（1 起始层号，逗号分隔）

    例如，将 Qwen3-8B 深度从 36 层剪到 24 层：

    PP=1 \
TARGET_NUM_LAYERS=24 \
HF_MODEL_CKPT=<pretrained_model_name_or_path> \
MLM_MODEL_SAVE=Qwen3-8B-Pruned \
./prune.sh qwen/Qwen3-8B

    [!TIP] 若模型层数无法被流水线并行度整除，可通过 MLM_EXTRA_ARGS="--decoder-first-pipeline-num-layers <X> --decoder-last-pipeline-num-layers <Y>" 设置不均匀 PP。
