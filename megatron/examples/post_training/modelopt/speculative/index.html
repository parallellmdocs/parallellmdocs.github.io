    推测解码

    # 推测式解码

    Medusa 与 EAGLE 的训练与模型导出均受支持（高速解码可通过 TensorRT-LLM 实现）。

    每步都会报告 Medusa 头部的 top-1 精度。注意：此处精度并非论文中的接受率，但第一个头的 top-1 能在一定程度上反映训练收敛情况。

    训练与导出流程

    实践中，应结合权重/kv-cache 量化以在不影响模型质量的前提下最大化 TPS。下文给出带自蒸馏的量化感知训练（QAT）流程。

    模型转换

    为避免质量下降，我们固定基座模型，仅将草稿模型作为附加组件。对于 Medusa，设置 --export-algorithm medusa 并提供 --export-num-medusa-heads；对于 EAGLE，设置 --export-algorithm eagle 并提供 --export-eagle-algorithm。转换后的 ${MLM_MODEL_SAVE} 将包含随机初始化的草稿模型权重。

    python examples/post_training/modelopt/convert_model.py \
    --export-algorithm eagle \
    --export-eagle-algorithm eagle3 \
    --load ${MLM_MODEL_CKPT} --save ${MLM_MODEL_SAVE} ${MLM_EXTRA_ARGS}

    合成数据生成

    草稿模型需要拟合基座模型的输出，因此自生成数据对提升精度和接受率至关重要。

    为简化流程，我们使用 vllm serve --quantization modelopt 部署量化模型，并以多轮对话数据作为输入生成回答。详细流程参考 ModelOpt 示例（https://github.com/NVIDIA/TensorRT-Model-Optimizer/tree/main/speculative_decoding）。生成结果以 OpenAI Chat Completion 格式的 jsonlines 保存。

    量化感知训练（QAT）

    QAT 流程包括 bf16 训练、模拟量化、QAT 微调。由于基座模型被冻结，初始训练主要用于估计草稿模型激活/权重范围。随后保存带量化因子的检查点，并继续在假量化模式下微调直至收敛。

    python examples/post_training/modelopt/finetune.py \
    --load ${MLM_MODEL_SAVE} --save ${MLM_MODEL_SAVE} ${MLM_EXTRA_ARGS}
python examples/post_training/modelopt/quantize.py \
    --export-quant-cfg fp8 \
    --load ${MLM_MODEL_SAVE} --save ${MLM_QUANT_SAVE} ${MLM_EXTRA_ARGS}
python examples/post_training/modelopt/finetune.py \
    --load ${MLM_QUANT_SAVE} --save ${MLM_QUANT_SAVE} ${MLM_EXTRA_ARGS}

    导出检查点

    最后导出 Medusa 头或 EAGLE 模块，以便在 TensorRT-LLM 等推理框架中部署。

    python examples/post_training/modelopt/export.py \
    --export-dir ${CKPT_DIR} \
    -export-extra-modules \
    --load ${MLM_QUANT_SAVE} ${MLM_EXTRA_ARGS}

    TensorRT-LLM 部署

    使用 TensorRT-LLM 服务导出后的检查点示例：

    trtllm-serve <exported checkpoint> --host 0.0.0.0 --port 8000 --backend pytorch --max_batch_size 32 --max_num_tokens 8192 --max_seq_len 8192 --tp_size 8 --extra_llm_api_options extra-llm-api-config.yml

    extra-llm-api-config.yml 示例：

    enable_attention_dp: false
disable_overlap_scheduler: true
enable_autotuner: false

cuda_graph_config:
    max_batch_size: 1

speculative_config:
    decoding_type: Eagle
    max_draft_len: 3
    speculative_model_dir: <eagle3 checkpoint>

kv_cache_config:
    enable_block_reuse: false
