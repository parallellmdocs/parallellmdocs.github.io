    MIMO：多模态输入/输出模型

    什么是 MIMO？

    MIMO 是一种帮助语言模型理解与生成多种模态（文本、图像、音频等）的架构，其核心思路包括：

    • 一个处理统一嵌入的语言模型
    • 模态特定的子模块，用于：
    • 将原始输入编码成嵌入（如图像 → embeddings）
    • 将嵌入解码回输出（如 embeddings → 图像）
    • 在模态空间与语言模型空间之间投射
    • MimoModel 负责：
    • 在序列中的特定 token 位置对齐模态嵌入
    • 将组合后的嵌入送入语言模型处理

    通过灵活的配置，MIMO 可扩展为多种多模态模型，例如：

    • 视觉语言模型（VLM）
    • 音频视觉语言模型
    • 多模态理解与生成模型

    工作原理

    整体架构包含两大组件：

    1. 语言模型
    2. 模态子模块

    完整的数据流如下：

    Input → Encoder → Projection → Align input embeddings → Language Model → Hidden states for special generation tokens ->  Output Projection → Decoder → Output
    1. 编码阶段：
    2. 模态子模块将输入转换为嵌入（例如图像 → embeddings）。
    3. MimoModel 将所有模态嵌入与文本嵌入按照 token 位置对齐。

    4. 语言模型处理统一后的嵌入序列。

    5. 解码阶段：

    6. 选择对应特定模态生成 token 的隐藏状态。
    7. 模态子模块将嵌入还原为输出（例如 embeddings → 图像）。

    组件详解

    语言模型

    语言模型是处理各模态信息的核心，负责：

    • 在共享词表中作为所有模态的中央处理器
    • 处理包含文本与模态 token 的混合序列

    模态子模块

    ModalitySubmodules 负责连接原始模态数据与语言模型：

    • 每个子模块都处理 编码（模态 → embeddings）与 解码（embeddings → 模态）
    • 管理模态空间与语言模型维度之间的 投射
    # Base class constructor with named encoders and decoders
class ModalitySubmodules(ABC, nn.Module):
    def __init__(
        self,
        encoders: Optional[Dict[str, nn.Module]] = None,
        decoders: Optional[Dict[str, nn.Module]] = None,
        input_projections: Optional[List[nn.Module]] = None,
        output_projections: Optional[List[nn.Module]] = None,
    ):

    MIMO 提供默认实现（VisionModalitySubmodules、AudioModalitySubmodules），也可以根据需求自定义：

    # Custom implementation example
class CustomVisionSubmodules(ModalitySubmodules):
    def encode(self, inputs):
        # Custom encoding logic
        return projected_embeddings

# Use custom submodules when building the model
model = MimoModel(
    mimo_config,
    modality_submodules={"images": ModuleSpec(module=CustomVisionSubmodules, params={...})}
)

    嵌入对齐

    MimoModel 的 align_embeddings_by_token_positions 方法负责整合不同模态的嵌入：

    • 根据特殊 token 位置将模态嵌入插入输入序列
    • 处理维度匹配与位置追踪

    内部示例：

    # Internal logic inside MimoModel forward
aligned_embeddings = self.align_embeddings_by_token_positions(
    modality_embeddings={"text": text_emb, "images": image_emb},
    input_ids=tokens,
    special_token_ids={"images": 32000}
)

    配置与使用

    MimoModel 参数

    MimoModel(
    config: MimoModelConfig,    # Required model configuration
)

    配置详情

    MimoModelConfig 包含以下内容：

    1. 语言模型的配置
    2. 模态名称到子模块配置的映射
    MimoModelConfig(
    language_model: ModuleSpec,                         # Specification for the language model
    modality_submodules: Dict[str, ModuleSpec],         # Modality name to submodule specification
    special_token_ids: Dict[str, int] = {}              # Special token IDs for each modality
)

    示例：构建视觉语言模型（VLM）

    # Language model specification
lm_spec = ModuleSpec(
    module=GPTModel,
    params={
        "config": language_config,
        "transformer_layer_spec": get_mock_language_layer_spec(),
        "vocab_size": 50304,
    }
)

# Vision modality specification
vision_submodule_spec = ModuleSpec(
    module=VisionModalitySubmodules,
    params={
        # Optional general parameters for the submodule
    },
    submodules={
        "encoders": {
            "clip_encoder": ModuleSpec(
                module=CLIPViTModel,
                params={
                    "transformer_config": vision_config,
                    "transformer_layer_spec": get_mock_vision_layer_spec(),
                    "patch_dim": 16,
                    "img_h": 224,
                    "img_w": 224,
                }
            ),
        },
        "input_projections": [
            ModuleSpec(
                module=MultimodalProjector,
                params={
                    "config": get_mock_projection_config(),
                    "submodules": get_mock_projection_layer_spec().submodules,
                    "projector_type": "mlp",
                    "input_size": 128
                }
            ),
        ],
    }
)

# Instantiate the model
vlm = MimoModel(
    MimoModelConfig(
        language_model=lm_spec,
        modality_submodules={"images": vision_submodule_spec},
        special_token_ids={"images": 32000}
    )
)

    MIMO 前向调用示例

    # Prepare multimodal inputs; keys must match the initialization config
modality_inputs = {
    "images": {
        "clip_encoder": {"pixel_values": images},  # Encoder-specific input
        "vit_encoder": {"images": vit_images}
    },
    "audio": {
        "whisper_encoder": {"input_features": audio_features}
    }
}

# 调用 forward
outputs, _ = mimo_model(
    input_ids=input_ids,
    position_ids=position_ids,
    modality_inputs=modality_inputs,
)
