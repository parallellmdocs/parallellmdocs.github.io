    深度估计

    深度估计数据集用于训练模型推断图像中每个像素到摄像头的相对距离。此类数据集主要应用于视觉感知与机器人感知，例如为自动驾驶绘制道路地图。本指南演示如何对深度估计数据集执行变换。

    开始前，请确保已安装最新版 albumentations：

    pip install -U albumentations

    Albumentations 是专为计算机视觉设计的数据增强库，支持图像分类、目标检测、分割、关键点估计等任务。

    本文使用 NYU Depth V2 数据集，该数据集包含由 RGB 与深度摄像头拍摄的室内视频序列，覆盖 3 座城市的场景，并同时提供图像与深度图作为标签。

    加载 train 划分并查看示例：

    >>> from datasets import load_dataset

>>> train_dataset = load_dataset("sayakpaul/nyu_depth_v2", split="train")
>>> index = 17
>>> example = train_dataset[index]
>>> example
{'image': <PIL.PngImagePlugin.PngImageFile image mode=RGB size=640x480>,
 'depth_map': <PIL.TiffImagePlugin.TiffImageFile image mode=F size=640x480>}

    数据集中包含两列：

    • image：PIL PNG 图像对象，数据类型为 uint8。
    • depth_map：PIL TIFF 图像对象，数据类型为 float32，即对应的深度图。

    深度图使用 TIFF 格式是因为它支持包括 float32 在内的多种数据类型；而 JPEG/PNG 只支持 uint8 或 uint16。如果你的深度图保存为 JPEG/PNG，可以使用 Image(mode="F") 将其读取为单通道的 float32：

    >>> from datasets import Image

>>> train_dataset = train_dataset.cast_column("depth_map", Image(mode="F"))

    查看原始图像：

    >>> example["image"]

    由于 PIL 无法直接显示 float32 图像，可先调用 .convert("RGB") 转为 uint8 后查看深度图：

    >>> example["depth_map"].convert("RGB")

    结果几乎全黑，需要对深度图进行着色才能直观展示。我们可以在显示时使用 plt.imshow() 自动着色，或借助 plt.cm 生成彩色深度图再显示。下面示例采用后者，便于后续保存（工具函数来自 FastDepth 仓库）：

    >>> import numpy as np
>>> import matplotlib.pyplot as plt

>>> cmap = plt.cm.viridis

>>> def colored_depthmap(depth, d_min=None, d_max=None):
...     if d_min is None:
...         d_min = np.min(depth)
...     if d_max is None:
...         d_max = np.max(depth)
...     depth_relative = (depth - d_min) / (d_max - d_min)
...     return 255 * cmap(depth_relative)[:, :, :3]

>>> def show_depthmap(depth_map):
...     if not isinstance(depth_map, np.ndarray):
...         depth_map = np.array(depth_map)
...     if depth_map.ndim == 3:
...         depth_map = depth_map.squeeze()
...
...     d_min = np.min(depth_map)
...     d_max = np.max(depth_map)
...     depth_map = colored_depthmap(depth_map, d_min, d_max)
...
...     plt.imshow(depth_map.astype("uint8"))
...     plt.axis("off")
...     plt.show()

>>> show_depthmap(example["depth_map"])

    还可以同时可视化多组图像及对应深度图：

    >>> def merge_into_row(input_image, depth_target):
...     if not isinstance(input_image, np.ndarray):
...         input_image = np.array(input_image)
...
...     d_min = np.min(depth_target)
...     d_max = np.max(depth_target)
...     depth_target_col = colored_depthmap(depth_target, d_min, d_max)
...     img_merge = np.hstack([input_image, depth_target_col])
...
...     return img_merge

>>> random_indices = np.random.choice(len(train_dataset), 9).tolist()
>>> plt.figure(figsize=(15, 6))
>>> for i, idx in enumerate(random_indices):
...     example = train_dataset[idx]
...     ax = plt.subplot(3, 3, i + 1)
...     image_viz = merge_into_row(example["image"], example["depth_map"])
...     plt.imshow(image_viz.astype("uint8"))
...     plt.axis("off")

    接下来使用 albumentations 应用数据增强，包括：

    • 随机水平翻转
    • 随机裁剪
    • 随机亮度对比度
    • 随机 Gamma
    • 随机色调饱和度
    >>> import albumentations as A

>>> crop_size = (448, 576)
>>> transforms = [
...     A.HorizontalFlip(p=0.5),
...     A.RandomCrop(crop_size[0], crop_size[1]),
...     A.RandomBrightnessContrast(),
...     A.RandomGamma(),
...     A.HueSaturationValue()
... ]

    通过 additional_targets 将目标键名映射得更直观：

    >>> additional_targets = {"depth": "mask"}
>>> aug = A.Compose(transforms=transforms, additional_targets=additional_targets)

    这样即可将深度图传给 depth 参数，而非默认的 mask，下方的 apply_transforms() 函数即体现了这一点。

    定义函数同时变换图像与深度图：

    >>> def apply_transforms(examples):
...     transformed_images, transformed_maps = [], []
...     for image, depth_map in zip(examples["image"], examples["depth_map"]):
...         image, depth_map = np.array(image), np.array(depth_map)
...         transformed = aug(image=image, depth=depth_map)
...         transformed_images.append(transformed["image"])
...         transformed_maps.append(transformed["depth"])
...
...     examples["pixel_values"] = transformed_images
...     examples["labels"] = transformed_maps
...     return examples

    使用 [~Dataset.set_transform] 在批量加载时按需应用转换，节省磁盘空间：

    >>> train_dataset.set_transform(apply_transforms)

    通过索引 pixel_values 与 labels 验证转换结果：

    >>> example = train_dataset[index]

>>> plt.imshow(example["pixel_values"])
>>> plt.axis("off")
>>> plt.show()

    再查看对应的深度图：

    >>> show_depthmap(example["labels"])

    也可以重复使用之前的 random_indices，可视化多组增广样本：

    >>> plt.figure(figsize=(15, 6))

>>> for i, idx in enumerate(random_indices):
...     ax = plt.subplot(3, 3, i + 1)
...     example = train_dataset[idx]
...     image_viz = merge_into_row(example["pixel_values"], example["labels"])
...     plt.imshow(image_viz.astype("uint8"))
...     plt.axis("off")
