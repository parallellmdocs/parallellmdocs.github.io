    快速上手

    • :material-rocket-launch: 在线体验快速上手示例 --- 在浏览器中运行完整 Notebook，立即熟悉 🤗 Datasets 的核心用法。
      :material-launch: 打开 Google Colab

    本快速上手指南面向已经准备直接动手编码的开发者，展示如何将 🤗 Datasets 集成到模型训练流程中。如果你刚刚接触这个库，建议先阅读我们的系列教程，那里提供了更系统的背景介绍。

    每个数据集都有自己的特点，根据任务类型的不同，训练前可能需要额外的数据准备步骤。但无论如何，你都可以依托 🤗 Datasets 提供的工具来加载并处理数据。启动最快的方式，是直接从 Hugging Face Hub 加载已有的数据集。Hub 上提供了覆盖多种任务的成千上万个数据集，选定你想要的数据类型，我们就开始吧！

    • 音频：对音频数据集进行重采样，让模型能够识别来电者所描述的银行问题。
    • 视觉：对图像数据执行增强，帮助模型诊断豆叶病害。
    • NLP：对文本数据进行分词，判断句子对是否表达相同含义。

    提示

    建议阅读 Hugging Face 课程的第 5 章，了解更多主题，例如加载远程或本地数据集、清理数据集的工具及如何创建自定义数据集。

    首先安装 🤗 Datasets：

    pip install datasets

    🤗 Datasets 同样支持音频与图像格式：

    • 若要处理音频数据集，请安装 [Audio] 特性：
    pip install datasets[audio]
    • 若要处理图像数据集，请安装 [Image] 特性：
    pip install datasets[vision]

    除了 🤗 Datasets，还需要安装你偏好的机器学习框架：

    bash pip install torch bash pip install tensorflow

    音频

    音频数据集的加载方式与文本数据集类似，但预处理方式有所不同。你需要使用特征提取器而不是分词器。根据预训练模型的要求，音频输入可能还需要重采样，以匹配模型的采样率。本节示例将准备 MInDS-14 数据集，以训练一个分类模型来判断用户遇到的银行问题。

    1. 通过 [load_dataset] 函数提供数据集名称、配置（并非所有数据集都有配置）以及数据划分来加载 MInDS-14：

    >>> from datasets import load_dataset, Audio

>>> dataset = load_dataset("PolyAI/minds14", "en-US", split="train")

    2. 使用 🤗 Transformers 库加载一个预训练的 Wav2Vec2 模型及其对应的特征提取器。加载后若看到权重未初始化的警告是正常的，因为我们会将该检查点用于另一个任务进行训练。

    >>> from transformers import AutoModelForAudioClassification, AutoFeatureExtractor

>>> model = AutoModelForAudioClassification.from_pretrained("facebook/wav2vec2-base")
>>> feature_extractor = AutoFeatureExtractor.from_pretrained("facebook/wav2vec2-base")

    3. MInDS-14 的数据卡指出其采样率为 8kHz，而 Wav2Vec2 模型是在 16kHz 的采样率下预训练的。需要结合 [~Dataset.cast_column] 函数与 [Audio] 特性，将 audio 列上采样至模型要求的采样率。

    >>> dataset = dataset.cast_column("audio", Audio(sampling_rate=16000))
>>> dataset[0]["audio"]
<datasets.features._torchcodec.AudioDecoder object at 0x11642b6a0>

    4. 编写一个预处理函数，使用特征提取器处理音频 array，并通过截断与填充生成等长的张量。务必记得将音频 array 传给特征提取器，因为 array（也就是语音信号本身）才是模型的输入。

    准备好预处理函数后，调用 [~Dataset.map] 对数据集进行批量处理，加速转换流程。

    >>> def preprocess_function(examples):
...     audio_arrays = [x.get_all_samples().data for x in examples["audio"]]
...     inputs = feature_extractor(
...         audio_arrays,
...         sampling_rate=16000,
...         padding=True,
...         max_length=100000,
...         truncation=True,
...     )
...     return inputs

>>> dataset = dataset.map(preprocess_function, batched=True)

    5. 使用 [~Dataset.rename_column] 将 intent_class 列重命名为 labels。在 Wav2Vec2ForSequenceClassification 中，模型期望输入列名称为 labels：

    >>> dataset = dataset.rename_column("intent_class", "labels")

    6. 根据所使用的机器学习框架设置数据集的格式。

    借助 [~Dataset.set_format] 将数据集格式设置为 torch，并指定需要转换的列。此函数会按需执行格式转换。完成 PyTorch 张量转换后，将数据集封装到 torch.utils.data.DataLoader 中：

    >>> from torch.utils.data import DataLoader

>>> dataset.set_format(type="torch", columns=["input_values", "labels"])
>>> dataloader = DataLoader(dataset, batch_size=4)

    使用 🤗 Transformers 提供的 [~transformers.TFPreTrainedModel.prepare_tf_dataset] 方法，将数据集封装为兼容 TensorFlow 的格式。该方法会将 [~datasets.Dataset] 转为 tf.data.Dataset，并自动处理整理与批次划分，因此可以直接传给 Keras 的 fit() 等方法。

    >>> import tensorflow as tf

>>> tf_dataset = model.prepare_tf_dataset(
...     dataset,
...     batch_size=4,
...     shuffle=True,
... )

    7. 现在就可以在你偏好的框架中开始训练了！想要获得端到端示例，请阅读 🤗 Transformers 的音频分类指南。

    视觉

    图像数据集的加载方式与文本数据集相同，只不过预处理阶段需要用到特征提取器。在计算机视觉任务中，应用数据增强有助于缓解过拟合。你可以选择任意数据增强库，并配合 🤗 Datasets 应用这些变换。本示例将加载 Beans 数据集，帮助模型学会识别叶片上的病害。

    1. 调用 [load_dataset] 并传入数据集名称与划分：

    >>> from datasets import load_dataset, Image

>>> dataset = load_dataset("AI-Lab-Makerere/beans", split="train")

    大多数视觉模型期望输入为 RGB 图像。如果你的数据集使用其他模式，可借助 [~Dataset.cast_column] 将其转换为 RGB：

    >>> dataset = dataset.cast_column("image", Image(mode="RGB"))

    Beans 数据集本身就是 RGB 图像，因此这里仅作演示。

    2. 选择你喜欢的数据增强库（如 Albumentations、imgaug、Kornia）。本示例借助 torchvision 随机调整图像的颜色属性：

    >>> from torchvision.transforms import Compose, ColorJitter, ToTensor

>>> jitter = Compose(
...     [ColorJitter(brightness=0.5, hue=0.5), ToTensor()]
... )

    3. 定义一个函数，对数据集中每张图像应用上述变换，并生成模型所需的 pixel_values：

    >>> def transforms(examples):
...     examples["pixel_values"] = [jitter(image.convert("RGB")) for image in examples["image"]]
...     return examples

    4. 使用 [~Dataset.with_transform] 在加载样本时按需应用数据增强：

    >>> dataset = dataset.with_transform(transforms)

    5. 根据你的训练框架设置数据格式。

    为 PyTorch 创建一个整理函数，将样本堆叠成批次，再交给 torch.utils.data.DataLoader：

    >>> from torch.utils.data import DataLoader

>>> def collate_fn(examples):
...     images = []
...     labels = []
...     for example in examples:
...         images.append(example["pixel_values"])
...         labels.append(example["labels"])
...
...     pixel_values = torch.stack(images)
...     labels = torch.tensor(labels)
...     return {"pixel_values": pixel_values, "labels": labels}
>>> dataloader = DataLoader(dataset, collate_fn=collate_fn, batch_size=4)

    借助 🤗 Transformers 的 [~transformers.TFPreTrainedModel.prepare_tf_dataset] 方法，将数据集包装成 tf.data.Dataset，即可结合 TensorFlow 训练或微调模型。运行前请确保已安装最新版 albumentations 与 cv2：

    pip install -U albumentations opencv-python

    >>> import albumentations
>>> import numpy as np

>>> transform = albumentations.Compose([
...     albumentations.RandomCrop(width=256, height=256),
...     albumentations.HorizontalFlip(p=0.5),
...     albumentations.RandomBrightnessContrast(p=0.2),
... ])

>>> def transforms(examples):
...     examples["pixel_values"] = [
...         transform(image=np.array(image))["image"] for image in examples["image"]
...     ]
...     return examples

>>> dataset.set_transform(transforms)
>>> tf_dataset = model.prepare_tf_dataset(
...     dataset,
...     batch_size=4,
...     shuffle=True,
... )

    6. 一切准备就绪，可以用你偏好的框架开始训练。若想查看端到端范例，请阅读 🤗 Transformers 的图像分类指南。

    NLP

    文本数据同样需要经过分词器处理成模型可接受的 token。本节示例将加载 Microsoft Research Paraphrase Corpus (MRPC) 训练集，训练一个模型来判断句子对是否表达相同含义。

    1. 调用 [load_dataset] 并提供数据集名称、配置（并非所有数据集都有配置）以及数据划分：

    >>> from datasets import load_dataset

>>> dataset = load_dataset("nyu-mll/glue", "mrpc", split="train")

    2. 加载预训练的 BERT 模型及对应分词器。加载时若出现权重未初始化的提示同样可以忽略，因为我们会把该检查点迁移到新任务上继续训练。

    >>> from transformers import AutoModelForSequenceClassification, AutoTokenizer

>>> model = AutoModelForSequenceClassification.from_pretrained("bert-base-uncased")
>>> tokenizer = AutoTokenizer.from_pretrained("bert-base-uncased")
===PT-TF-SPLIT===
>>> from transformers import TFAutoModelForSequenceClassification, AutoTokenizer

>>> model = TFAutoModelForSequenceClassification.from_pretrained("bert-base-uncased")
>>> tokenizer = AutoTokenizer.from_pretrained("bert-base-uncased")

    3. 编写一个函数执行分词，同时对文本进行截断与填充，以生成规则的张量。分词器会在数据集中新增 input_ids、token_type_ids 与 attention_mask 三列，分别对应模型输入。

    使用 [~Dataset.map] 将分词函数应用到数据集的批量样本上，以加速处理：

    >>> def encode(examples):
...     return tokenizer(
...         examples["sentence1"],
...         examples["sentence2"],
...         truncation=True,
...         padding="max_length",
...     )

>>> dataset = dataset.map(encode, batched=True)
>>> dataset[0]

    4. 将 label 列映射为模型期望的 labels：

    >>> dataset = dataset.map(lambda examples: {"labels": examples["label"]}, batched=True)

    5. 按所用框架设置数据格式。

    借助 [~Dataset.with_format] 将数据集设置为 torch 格式，并只保留模型所需的列。随后使用 torch.utils.data.DataLoader 迭代数据：

    >>> import torch

>>> dataset = dataset.select_columns(["input_ids", "token_type_ids", "attention_mask", "labels"])
>>> dataset = dataset.with_format(type="torch")
>>> dataloader = torch.utils.data.DataLoader(dataset, batch_size=32)

    继续使用 [~transformers.TFPreTrainedModel.prepare_tf_dataset] 准备 TensorFlow 训练所需的数据集，并按需指定批大小与是否打乱数据：

    >>> import tensorflow as tf

>>> tf_dataset = model.prepare_tf_dataset(
...     dataset,
...     batch_size=4,
...     shuffle=True,
... )

    6. 万事俱备。立刻用你熟悉的框架开始训练吧！完整示例可参考 🤗 Transformers 的文本分类指南。
