    数据处理

    🤗 Datasets 提供大量工具用于调整数据集的结构与内容，例如清洗数据、创建新列、在不同特征或格式之间转换等。

    本指南将介绍如何：

    • 重排行顺序并划分数据集。
    • 重命名、删除列以及常见列操作。
    • 对每条样本执行处理函数。
    • 拼接多个数据集。
    • 设置自定义输出格式。
    • 保存与导出处理后的数据。

    若想了解特定模态的处理方式，请参考音频处理、图像处理与文本处理指南。

    示例使用 MRPC 数据集，你也可以替换为其他数据集：

    >>> from datasets import load_dataset
>>> dataset = load_dataset("nyu-mll/glue", "mrpc", split="train")

    注意

    以下处理方法都会返回新的 [Dataset] 对象，不会在原地修改。请注意不要覆盖原始数据。

    排序、打乱、筛选、划分与分片

    这些函数可帮助你挑选所需行、创建训练/测试划分，或将超大数据集拆分为小块。

    排序

    [~Dataset.sort] 会根据列的数值排序，该列需兼容 NumPy：

    >>> sorted_dataset = dataset.sort("label")

    底层通过重排索引列表实现。

    打乱

    [~Dataset.shuffle] 随机打乱行顺序，可通过 generator 自定义随机数生成器：

    >>> shuffled_dataset = sorted_dataset.shuffle(seed=42)

    注意：一旦数据集拥有索引映射（如执行了 sort 或 shuffle），读取速度可能下降，因为需要额外步骤定位行且不再是连续块。可通过 [Dataset.flatten_indices] 重写数据恢复速度，或转为 [IterableDataset] 并使用其快速近似打乱 [IterableDataset.shuffle]：

    >>> iterable_dataset = dataset.to_iterable_dataset(num_shards=128)
>>> shuffled_iterable_dataset = iterable_dataset.shuffle(seed=42, buffer_size=1000)

    选择与过滤

    [~Dataset.select] 根据索引选择行：

    >>> small_dataset = dataset.select([0, 10, 20, 30, 40, 50])

    [~Dataset.filter] 根据条件筛选行：

    >>> start_with_ar = dataset.filter(lambda example: example["sentence1"].startswith("Ar"))

    设置 with_indices=True 可在过滤时使用索引：

    >>> even_dataset = dataset.filter(lambda example, idx: idx % 2 == 0, with_indices=True)

    划分

    [~Dataset.train_test_split] 可快速生成训练/测试划分，并指定比例或样本数量：

    >>> dataset.train_test_split(test_size=0.1)

    默认会先打乱，可通过 shuffle=False 关闭。

    分片

    [~Dataset.shard] 支持将大数据集拆分为若干份，需指定 num_shards 与返回的 index：

    >>> dataset.shard(num_shards=4, index=0)

    重命名、删除、转换与展开列

    重命名

    使用 [~Dataset.rename_column] 修改列名，原列的特征会移动至新列：

    >>> dataset = dataset.rename_column("sentence1", "sentenceA")

    删除

    [~Dataset.remove_columns] 删除一个或多个列：

    >>> dataset = dataset.remove_columns(["sentence1", "sentence2"])

    [~Dataset.select_columns] 与之相反：保留指定列并移除其他列。

    类型转换

    [~Dataset.cast] 可一次性转换多列的特征类型，需要提供新的 [Features]：

    >>> new_features = dataset.features.copy()
>>> new_features["label"] = ClassLabel(names=["negative", "positive"])
>>> dataset = dataset.cast(new_features)

    提示

    仅当原始类型与目标类型兼容时才可转换，例如由 Value("int32") 转为 Value("bool") 需保证原列仅包含 0 与 1。

    若只需转换单列，可使用 [~Dataset.cast_column]。

    展开嵌套

    对于嵌套字段，可使用 [~Dataset.flatten] 将子字段展开为独立列。例如 SQuAD 数据集中的 answers 字段：

    >>> flat_dataset = dataset.flatten()

    map：逐样本或批量处理

    [~Dataset.map] 是最强大的工具之一，可在单条或批量样本上执行函数，甚至生成新列或新行。

    基本示例

    为 sentence1 添加前缀：

    >>> def add_prefix(example):
...     example["sentence1"] = "My sentence: " + example["sentence1"]
...     return example
>>> updated_dataset = dataset.map(add_prefix)

    可通过 remove_columns 删除列；设置 with_indices=True 可以同时使用索引。

    多进程

    通过 num_proc 并行处理可显著提速：

    >>> updated_dataset = dataset.map(func, num_proc=4)

    with_rank=True 可在分布式环境中使用进程 rank，常用于多 GPU 并行。注意使用 multiprocess.set_start_method("spawn")，否则会出现 CUDA 错误。

    分批处理

    设置 batched=True 可以按批处理，默认批大小为 1000，可通过 batch_size 调整。

    拆分长文本

    >>> def chunk_examples(examples):
...     chunks = []
...     for sentence in examples["sentence1"]:
...         chunks += [sentence[i:i + 50] for i in range(0, len(sentence), 50)]
...     return {"chunks": chunks}
>>> chunked_dataset = dataset.map(chunk_examples, batched=True, remove_columns=dataset.column_names)

    数据增强

    结合模型生成替换词进行增广：

    >>> from transformers import pipeline
>>> fillmask = pipeline("fill-mask", model="roberta-base")
>>> def augment_data(examples):
...     outputs = []
...     for sentence in examples["sentence1"]:
...         # ...略
...     return {"data": outputs}
>>> augmented_dataset = smaller_dataset.map(augment_data, batched=True, remove_columns=dataset.column_names, batch_size=8)

    异步处理

    可定义 async/await 函数并在 map 中调用，以并行访问 API（此处略，逻辑同英文示例）。

    拼接数据集

    使用 [datasets.concatenate_datasets] 纵向拼接、[datasets.interleave_datasets] 交错合并，或直接构建 [DatasetDict]。

    格式化输出

    set_format / with_format

    [~Dataset.set_format] 会就地设定输出格式；[~Dataset.with_format] 返回应用格式后的新数据集。

    支持格式包括：

    • 张量/数组："numpy"、"torch"、"tensorflow"、"jax"
    • 表格："pandas"、"polars"、"arrow"

    示例：

    >>> ds = Dataset.from_dict({"text": ["foo", "bar"], "tokens": [[0,1,2],[3,4,5]]})
>>> ds = ds.with_format("torch")
>>> ds[0]
{'text': 'foo', 'tokens': tensor([0, 1, 2])}

    使用表格格式可避免数据复制，并在 map/filter 中提升性能：

    >>> ds = ds.with_format("pandas")
>>> ds = ds.map(lambda df: df.assign(upper_text=df.text.str.upper()), batched=True)

    自定义格式

    [~Dataset.with_transform] 可在访问样本时即时执行转换，例如动态分词与填充：

    >>> tokenizer = AutoTokenizer.from_pretrained("bert-base-uncased")
>>> def encode(batch):
...     return tokenizer(batch["sentence1"], batch["sentence2"], padding="longest", truncation=True, max_length=512, return_tensors="pt")
>>> dataset = dataset.with_transform(encode)

    也可用于自定义解码。例如以下代码使用 pydub 读取音频（参见英文示例）。

    保存与导出

    处理完成后，可通过以下方法保存：

    • [~Dataset.save_to_disk] / [~Dataset.load_from_disk]
    • 导出为 CSV/JSON/Parquet/SQL（to_csv、to_json 等）
    • [~Dataset.push_to_hub] 上传到 Hub

    也可以使用 [~Dataset.save_faiss_index]、[~Dataset.save_elasticsearch_index] 等保存索引。
