    搭配 NumPy 使用

    本文介绍如何从 datasets 获取 numpy.ndarray，并将其用于基于 NumPy 的模型（如 scikit-learn）。

    设置数据集格式

    默认情况下，数据集返回 Python 原生对象。若希望返回 NumPy 数组，可设置格式为 numpy：

    from datasets import Dataset
data = [[1, 2], [3, 4]]
ds = Dataset.from_dict({"data": data}).with_format("numpy")
ds[0]   # {'data': array([1, 2])}

    同样适用于 DatasetDict：

    dds = DatasetDict.from_dict({"train": {"data": [[1, 2]]}})
dds = dds.with_format("numpy")

    多维数组

    若数组形状固定，会自动拼接为同一张量；若形状不固定则回退为 dtype=object。建议在定义特征时使用 [Array2D] 等明确声明形状以避免额外复制：

    features = Features({"data": Array2D(shape=(2, 2), dtype="int32")})

    其他特征类型

    [ClassLabel] 会被正确转换为数组；字符串及二进制保持不变。[Image]、[Audio] 也受支持（需安装 datasets[vision] 或 datasets[audio]）：

    features = Features({"image": Image()})
ds = Dataset.from_dict({"image": ["path/to/img.png"]}, features=features).with_format("numpy")
ds[0]["image"].shape  # (H, W, C)

    数据加载

    NumPy 本身不提供数据加载器，可通过 with_format('numpy') 直接获取数组，或结合 Dataset.iter() 按批迭代。以 MNIST 为例：

    ds = load_dataset("mnist").with_format("numpy")
for batch in ds["train"].iter(batch_size=32):
    x, y = batch["image"], batch["label"]

    若需更复杂的加载逻辑，可结合 PyTorch DataLoader 等框架使用。
