    安装

    在动手之前，需要准备好运行环境并安装相关依赖。🤗 Datasets 支持 Python 3.9 及更高版本。

    [!TIP] 若你计划结合 TensorFlow 或 PyTorch 使用 🤗 Datasets，请分别安装对应框架。具体安装命令请参考 TensorFlow 官方指南或 PyTorch 官网。

    虚拟环境

    建议在虚拟环境中安装 🤗 Datasets，以便隔离依赖、保持工作区整洁。

    1. 创建项目目录并切换至该目录：
    mkdir ~/my-project
cd ~/my-project
    1. 在目录内创建虚拟环境：
    python -m venv .env
    1. 通过以下命令激活或退出虚拟环境：
    # 激活虚拟环境
source .env/bin/activate

# 退出虚拟环境
source .env/bin/deactivate

    完成虚拟环境创建后，就可以在其中安装 🤗 Datasets 了。

    pip

    最简单的安装方式是使用 pip：

    pip install datasets

    执行下列命令确认安装是否成功：

    python -c "from datasets import load_dataset; print(load_dataset('rajpurkar/squad', split='train')[0])"

    上述命令会下载 Stanford Question Answering Dataset (SQuAD) 版本 1，加载训练集并打印首条样本。输出类似：

    音频

    若要处理音频数据集，请额外安装 [Audio] 特性：

    pip install datasets[audio]

    视觉

    若要处理图像数据集，请额外安装 [Image] 特性：

    pip install datasets[vision]

    源码安装

    如果你希望自定义或修改 🤗 Datasets 的代码，可以从源码构建。克隆仓库并执行以下命令：

    git clone https://github.com/huggingface/datasets.git
cd datasets
pip install -e .

    同样可通过前文的 Python 单行命令验证安装是否成功：

    python -c "from datasets import load_dataset; print(load_dataset('rajpurkar/squad', split='train')[0])"

    conda

    🤗 Datasets 也可以通过 conda 安装：

    conda install -c huggingface -c conda-forge datasets
