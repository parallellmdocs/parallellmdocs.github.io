    PyTorch API

    Transformer Engine 针对 PyTorch 提供了高性能模块、自动混合精度工具与辅助函数。本页按照模块类别列出可用接口；详细参数说明可在完整 API 生成流程中查看。

    核心层与模块

    • transformer_engine.pytorch.Linear：带有 FP8 缩放与张量并行分组接口的线性层。
    • transformer_engine.pytorch.GroupedLinear：针对专家并行/分组权重的线性层，实现高效块状 GEMM。
    • transformer_engine.pytorch.LayerNorm：TE 版本的 LayerNorm，保持与 PyTorch 接口一致。
    • transformer_engine.pytorch.RMSNorm：提供 RMSNorm 变体，用于替代标准 LayerNorm。
    • transformer_engine.pytorch.LayerNormLinear：在一次调用中完成 LayerNorm + 线性层的融合模块。
    • transformer_engine.pytorch.LayerNormMLP：包含 LayerNorm 的前馈层组合，适合 GPT/T5 结构。

    注意力与 Transformer 组件

    • transformer_engine.pytorch.DotProductAttention：支持上下文并行的点积注意力实现。
    • transformer_engine.pytorch.MultiheadAttention：TE 优化的标准多头注意力，可同时配置张量并行与上下文并行组。
    • transformer_engine.pytorch.TransformerLayer：整合注意力、前馈与归一化的 Transformer 基础层。
    • transformer_engine.pytorch.dot_product_attention.inference.InferenceParams：推理阶段用于管理 KV 缓存和步长的辅助结构。

    随机数与缓冲区管理

    • transformer_engine.pytorch.CudaRNGStatesTracker：跟踪 CUDA RNG 状态，便于在不同阶段复现随机数。
    • transformer_engine.pytorch.UserBufferQuantizationMode：枚举 UserBuffer 支持的量化模式（FP8、NONE）。

    自动混合精度与模型初始化

    • transformer_engine.pytorch.fp8_autocast / autocast：上下文管理器，在前向传播中启用 FP8 自动混合精度。
    • transformer_engine.pytorch.fp8_model_init / quantized_model_init：以 FP8/BF16 等精度初始化模块参数。
    • transformer_engine.pytorch.checkpoint：集成 PyTorch 激活重计算机制，保证与 FP8 兼容。
    • transformer_engine.pytorch.is_fp8_available、is_mxfp8_available、is_fp8_block_scaling_available、is_nvfp4_available、is_bf16_available：快速检测当前硬件/软件是否支持对应数值格式。
    • transformer_engine.pytorch.get_cudnn_version、get_device_compute_capability：查询运行时环境。
    • transformer_engine.pytorch.get_default_recipe：返回默认的 FP8 缩放配方。
    • transformer_engine.pytorch.make_graphed_callables：基于 CUDA Graphs 封装可复用的前向/反向函数。
    • transformer_engine.pytorch.get_cpu_offload_context：获取与 CPU Offload 相关的上下文配置。

    MoE 相关工具

    • transformer_engine.pytorch.moe_permute / moe_unpermute：在专家前后执行张量重排。
    • transformer_engine.pytorch.moe_permute_with_probs / moe_sort_chunks_by_index_with_probs：同时保留路由概率的重排与排序函数。
    • transformer_engine.pytorch.moe_sort_chunks_by_index：根据专家索引对张量块排序。
    • transformer_engine.pytorch.parallel_cross_entropy：支持并行专家路由的交叉熵实现。

    UserBuffer 管理

    • transformer_engine.pytorch.initialize_ub：初始化 UserBuffer，用于在流水线/专家并行场景下缓存张量。
    • transformer_engine.pytorch.destroy_ub：释放 UserBuffer 资源。
